# AEGIS – Multi-Agent Capital & Trading System

A modular, production-grade multi-agent trading and capital management platform.

## Features
- Main Agent: Orchestrator, risk governor, capital allocator
- Market Agents: Forex, Indian Equity, Commodities, Long-term Investing
- Shared core modules for ML, risk, data, and logging
- Config-driven, extensible architecture

## Structure
See the `aegis/` directory for full module layout.

## Getting Started
- All modules are placeholders with docstrings and TODOs.
- No business logic is implemented yet.
- Designed for extensibility and production-readiness.

## Requirements
- Python 3.9+
- VS Code (recommended)
- GitHub Copilot (recommended)
